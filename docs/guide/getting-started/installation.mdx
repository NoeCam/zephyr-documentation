# Installation

import {Tabs, Tab} from '@theme';

## Generating a new project
When we started Zephyr Cloud we wanted it to be incredibly easy to get started.

If you do not already have a project to start with we suggest generating a new React project with [Nx](https://nx.dev/) and [Rspack](https://rspack.dev/).
<Tabs values={[{ label: 'Rspack' }, { label: 'webpack' }]}>
  <Tab>

    ```sh
    npx create-nx-workspace@latest --preset=@nx/rspack
    ```

  </Tab>

  <Tab>

    ```sh
    npx create-nx-workspace@latest
    ```

  </Tab>
</Tabs>

## Installing and Logging into Zephyr
The first step in getting started is to install the Zephyr Cloud Cli.

```sh filename="Terminal"
npx zephyr-cli login
```

## Modifying build configuration
